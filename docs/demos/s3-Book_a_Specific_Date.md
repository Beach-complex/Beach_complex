## 시나리오 3 지정된 날짜 예약하기
### 목표 :
사용자가 로그인 후 특정 해수욕장에 대해 원하는 날짜로 예약을 생성하고, 결과가 캘린더에 반영되는 것을 보여준다.

### 사전 조건
- jwt 로그인/세션 유지 가능
- 예약 생성 API/ 캘린더의 표시
- 최소 입력 검증:
- 날짜 필수
- 과거 날짜 불가
- 중복 예약 방지

### 데모 스크립트(사용자 액션)
1. 상세 화면에서 예약하기 클릭
2. 날짜 선택
3. 예약 제출
4. 성공확인
   - 캘린더의 해당 날짜 예약 완료됨
   - 예약 완료 알림이 발생함
5. 예외 케이스
   - 같은 날짜로 다시 예약 시도 시 이미 예약됨이 떠야한다.
   - 또는 과거 날짜 선택 -> 유효성 에러

### 성공 기준
로그인 사용자가 특정 해수욕장에 대해 유효한 미래 날짜로 예약을 생성하면 저장되고, 생성 직후 캘린더에서 예약 상태가 즉시 확인 가능해야한다.
